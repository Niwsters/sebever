How does personal development affect software development? I once had the notion that all it took to make great software was some programming knowledge. I learned Java to make games. I learned PHP, HTML, MySQL and CSS to make web applications. As I noticed quite quickly after learning them, making great software takes so much more. Anyone who has tried to make a game from scratch knows what I'm talking about - you get a great idea, start coding, make something work, and at the start of the next step there's an obstacle.
